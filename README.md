# nanf

Front-end for nan

## Installation

Download nanf.jar from [here](https://bitbucket.org/subaru45/nanf/downloads), and move it to the directory installed [nan](https://bitbucket.org/subaru45/nan).

## Usage

To run, double-click nanf.jar or type on command line:

    $ java -jar nanf.jar
